# 毛概选择题练习

在线毛概选择题练习系统，支持答题、收藏、错题重练等功能。

## 功能特点

- 📝 **180道选择题**：涵盖毛泽东思想和中国特色社会主义理论体系
- ✅ **即时反馈**：答题后立即显示对错和详细解析
- 🔄 **重新选择**：答错后可重新选择正确答案
- ⭐ **收藏功能**：可收藏重要题目，方便重点复习
- 📚 **多种模式**：
  - 全部题目
  - 随机20题
  - 错题重练
  - 收藏题目
  - 快速浏览
- 💾 **本地存储**：答题进度、错题、收藏自动保存

## 使用方法

直接打开 `index.html` 文件即可使用，无需服务器。

## 题目来源

- 毛概选择题知识点（90题）
- 毛概复习题（90题）

## 技术栈

- 纯前端实现，无需后端
- HTML + CSS + JavaScript
- 使用 localStorage 保存数据
